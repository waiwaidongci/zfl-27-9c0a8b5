# 古籍修补台

一个拖拽纸片归位、沿字列和边缘修复残页的静态前端拼字解谜游戏。

---

## 路径一：普通游玩（零依赖）

无需安装任何东西，直接双击打开即可游玩。

```
双击 index.html
```

或在浏览器地址栏输入 `file:///path/to/zfl-27/index.html`。

### 功能一览

- 拖拽纸片到正确格子，靠近时会自动吸附
- 支持存档迁移、每日挑战、残页编辑器
- 支持分享码（生成 / 导入）和关卡包（导出 / 导入）
- 完整支持触屏操作和键盘操作

---

## 路径二：开发验证（自动化测试）

如果需要在浏览器中跑通核心冒烟流程，请按照以下步骤操作。

### 环境要求

- Node.js >= 16

### 首次安装

```bash
cd zfl-27
npm install
npm run install:browsers
```

> `npm run install:browsers` 会下载 Playwright 所需的 Chromium 浏览器，只需执行一次。

### 一键启动静态服务

```bash
npm run serve
```

启动后访问 [http://localhost:8765](http://localhost:8765) 即可游玩（与直接打开 `index.html` 效果一致，区别在于此方式支持完整的浏览器自动化测试）。

### 运行全部测试

```bash
npm test
```

会依次执行：
1. **分享码纯逻辑测试**（Node.js 端，无需浏览器）：校验分享码编码 / 解码、截断、净化、安全过滤等逻辑
2. **浏览器冒烟测试**（Playwright + Chromium）：在真实浏览器中自动跑通核心流程

### 单独运行某类测试

```bash
# 只跑分享码纯逻辑测试
npm run test:sharecode

# 只跑浏览器冒烟测试
npm run test:smoke
```

### 浏览器冒烟测试覆盖的场景

| # | 场景 | 说明 |
|---|------|------|
| 1 | 进入游戏并完成一次放置 | 验证页面加载、残片渲染、拖拽放置、分数更新 |
| 2 | 打开编辑器并预览 | 验证编辑器入口、表单填充、预览试玩、返回游戏 |
| 3 | 生成并导入分享码 | 验证分享码导出、格式校验、解析预览、关闭弹窗 |
| 4 | 导入 test-pack.zflpack.json | 验证关卡包文件读取、导入流程启动 |

测试文件位于 [tests/smoke.spec.js](file:///Users/ali/Desktop/zfl%20%20new%20solo%20coder/zfl-27/tests/smoke.spec.js)。

## 本地验证说明（原有手工验证清单）

### 鼠标操作
1. 点击残片按住拖拽到目标位置
2. 靠近正确格子（42像素内）会自动吸附
3. 方向正确时会锁定并加分

### 键盘操作验证
| 按键 | 功能 | 验证步骤 |
|------|------|----------|
| Tab / 方向键 | 选择残片 | 1. 按 Tab 或任意方向键激活键盘模式（右上角出现绿色指示灯）<br>2. 用 ←→↑↓ 或 Tab 在残片间切换，绿色脉冲框为焦点 |
| 空格 / Enter | 确认选择 | 1. 焦点在目标残片上时按空格/回车<br>2. 残片变为选中状态，棋盘出现绿色虚线焦点框 |
| 方向键（选中后） | 移动目标格 | 1. 选中残片后用 ←→↑↓ 移动棋盘上的绿色虚线框<br>2. 目标格位置会平滑过渡 |
| 空格 / Enter（选中后） | 确认放置 | 1. 将虚线框移到目标位置后按空格/回车<br>2. 残片会移动到目标格并触发吸附判定<br>3. 位置和方向都正确时自动锁定 |
| R | 顺时针旋转 | 选中残片后按 R，残片顺时针旋转90度 |
| E | 逆时针旋转 | 选中残片后按 E，残片逆时针旋转90度 |
| F | 水平翻转 | 选中残片后按 F，残片水平翻转 |
| ESC | 取消选择 | 1. 按 ESC 取消当前选中<br>2. 回到残片选择模式 |

### 触屏操作验证
1. **防误触滚动**：用手指按住残片拖拽时，页面不会滚动
2. **拖拽流畅**：手指移动时残片跟随移动，松手时触发吸附判定
3. **点击反馈**：点击残片时有轻微缩小效果（scale 0.98）
4. **触控目标**：移动端残片最小尺寸为 44x44px，符合触控规范

### 移动端布局验证
- **950px以下**：三栏变单栏布局，按钮和文字适配缩小
- **480px以下**：工具箱变2列，状态栏变单栏
- **横屏/竖屏切换**：调整窗口大小检查布局自适应

### 操作状态协同验证
1. 选中残片后，工具箱同步显示当前选中信息
2. 放置成功后，自动取消选中并重置焦点到残片列表
3. 放置成功后，分数和错误次数实时更新
4. 键盘模式下所有操作与现有拖拽吸附判定完全一致

### 焦点样式验证
- 残片焦点：绿色脉冲边框（keyboard-pulse 动画）
- 目标格焦点：绿色虚线闪烁边框（cell-focus-blink 动画）
- 键盘模式指示：右上角绿色呼吸指示灯（keyboard-indicator 动画）
- 底部提示条：键盘模式下显示操作提示
